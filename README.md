# Tikitok Full

Это большой демо-клон TikTok без внешних зависимостей.

## Что есть

- полноэкранная лента
- нижняя навигация: Home / Discover / Studio / Inbox / Profile
- окно комментариев снизу
- вход и регистрация
- студия с загрузкой фото и видео
- быстрый клип без файла
- лайки, комментарии, профиль
- автообновление через обычный polling, без websocket и без SSE

## Локальный запуск

```bash
node server.js
```

Открой:

```text
http://localhost:10000
```

## Деплой

### Вручную

- Runtime: `Node`
- Build Command: `npm install`
- Start Command: `node server.js`

### Через Blueprint

В репозитории уже есть файл конфигурации `render.yaml`.

## Важное замечание

Файловая система на временных инстансах может быть эфемерной. Это значит:

- загруженные файлы
- JSON-база
- новые аккаунты
- новые посты

могут пропасть после рестарта, redeploy или spin down.

Для настоящего продакшена перенеси данные в базу и object storage.
